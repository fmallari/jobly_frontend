# Jobly Frontend

React + Vite frontend for Jobly — browse companies, search jobs, register/login, apply to jobs, and view your applications.

Connect to Jobly Backend -- https://github.com/fmallari/jobly.git

## Features
- Browse companies + company detail pages
- Browse/search jobs
- Authentication (register/login/logout) with JWT
- Apply to jobs (logged-in users)
- Applications page (protected route) to view applied jobs
- Clean UI with Tailwind CSS

## Tech Stack
- React (Vite)
- React Router
- Tailwind CSS
- Axios

## Getting Started

### Prerequisites
- Node.js (LTS recommended)
- Backend running locally (see backend README)

### Install
```bash
npm install
```
## Environment variables 
VITE_API_BASE_URL=http://localhost:3001

## Run app 
npm run dev
open: http://localhost:5173
